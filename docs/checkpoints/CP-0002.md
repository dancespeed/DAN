# Architectural decisions

---

## 2026-07-17

### Sequencer

Decision:
Sequencer publishes only generic completion event.

Reason:
Sequencer must remain independent from business logic.

Status:
Approved.

---

## 2026-07-18

### Actuator model

Decision:
Commands describe target state.

Example:

SetBrightness(channel,target,time)

Instead of

IncreaseBrightness()

Reason:
Allows interruption and deterministic behavior.

Status:
Approved.

## Rules
