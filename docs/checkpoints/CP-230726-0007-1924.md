## Совместимость

EventBus временно выполняет две функции одновременно:

```
Event
    ↓
EventDispatcher

Message
    ↓
MessageBus
    ↓
MessageDispatcher
```

То есть одно опубликованное событие одновременно попадает:

- в старый транспорт;
- в новый транспорт.

---

## Legacy-производители

- System

- Logic
- PWM

---

## Главный цикл

MessageDispatcher подключён после EventDispatcher.

Порядок обработки:

```
Input
    ↓
EventDispatcher
    ↓
MessageDispatcher
    ↓
PWM
```
