# Архитектурные решения

Утверждено.

MessageBus — namespace.

Dispatcher — отдельный компонент архитектуры.

Dispatcher не является переходником.

Dispatcher является постоянной частью платформы.

# Правила проекта

Работа ведется:

- архитектура сначала
- кирпичик за кирпичиком
- код только после утверждения архитектуры
- после каждого кирпичика проект должен собираться
- старая и новая архитектуры взаимодействуют только через адаптер
